Användarflöde för cypress test

1. Användaren går in på sidan och loggar in genom Github.
2. Användaren ser därefter formuläret för att skapa ett nytt inlägg.
3. Zod verifierar att användaren skriver i rätt typer av information i fälten.
4. Cypress testar att det som fyllts i i formuläret faktiskt renderas ut på sidan.
5. Cypress verifierar att ett error kastas om något gått fel.
